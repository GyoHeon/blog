## 팀 버너스리

팀 버너스리는 영국의 컴퓨터 과학자이다.
옥스퍼드 대학교 물리학 출신이라고 한다. (역시 물리는 대단하다.)

정말이지 현대 컴퓨너의 기초를 전부 다 만든 사람이라고 해도 과언이 아니다.
면접 준비할때 나오는 지식들의 상당수가 팀 버너스리가 만든 것들이다.
그래서 사람들을 힘들게 하는 분이다.

### 팀 버너스리가 만든 것들

1. www
2. http
3. html
4. uri
5. web server
