## 자잘한 리액트 이야기

리액트는 지금 서버인지 아닌지 어떻게 판단하는가?

`/shared/ExcutionEnvironment`

```ts
export const canUseDOM: boolean = !!(
  typeof window !== "undefined" &&
  typeof window.document !== "undefined" &&
  typeof window.document.createElement !== "undefined"
);
```

끝!
