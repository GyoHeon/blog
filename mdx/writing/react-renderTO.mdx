React의 컴포넌트들을 string이나 HTML로 활용할 때 좋은 내장 함수가 renderToString, renderToStaticMarkup이다.
둘의 차이점을 알아보고 직접 사용해보자.

### renderToStaticMarkup, renderToString

둘다 HTML 마크업 구조의 string이긴 한디
전자는 React Component를 HTML 마크업 구조로 변환하고 React와 관련된 attribute 등을 삭제해버린다.
후자는 React Component를 HTML 마크업 구조로 변환하고 React와 관련된 attribute 등을 남김.
그러나 그 문자 만으로는 정보가 없고 이를 클라이언트에서 받아서 액션을 취할수 있음.
