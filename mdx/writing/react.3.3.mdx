## Sharing State Between Components

두 컴포넌트 간에 완전히 동일하게 동작하는 state가 있다고 해보자.
이를 위해선 두 컴포넌트에 선언된 state를 가장 가까운 상위 컴포넌트에서 선언한 뒤 이를 props로 내려주는 방법이 있을 것이다.
이 방법은 _lifting state up_ 라고 불리며, React 에서 굉장히 흔한 패턴이다.
